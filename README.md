# Estructuras_de_datos-graduacion_unal_2026_01

Proyecto de clase del Grupo 2 para Estructuras de Datos (Grupo 2) en el semestre 2026-1. Con el objetivo de resolver el problema de "graduación unal" que consiste en indicar la duración en semestres y las materias a ver en un plan academico teniendo en cuenta los prerrequisitos de las asignaturas y la carga máxima por semestre.

## Instrucciones de instalación

Instalar y correr el proyecto en la consola de comandos requiere tener instalado:

- Maven 3.X
- Java 17+

Si se cumplen los requerimientos se puede correr el proyecto ejecutando en la carpeta raíz del proyecto los comandos:

```mvn package```
```java -jar target/graduacion_unal-1.0.jar```
